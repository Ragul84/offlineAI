## Full description
EdgeAI Tutor Lite is a privacy-first, fully offline educational companion built for students using budget Android phones.

Why students love EdgeAI Tutor Lite:
- 100% offline after one-time model download
- Camera-first learning: scan textbook pages, handwritten notes, and diagrams
- Voice-interactive chat for quick doubt solving
- Daily micro-quizzes and instant flashcards
- Progress dashboard with weak-area tips
- English, Tamil, and Hindi support
- No ads, no premium lock, no cloud dependence for tutoring

Built for real-world constraints:
- Runs on Android 11+ devices with 4-6 GB RAM
- Optimized for speed and battery-aware operation
- Optional enhanced model mode for deeper explanations

Privacy first:
- Learning content stays on device
- Analytics is optional and anonymous
- Easy local data reset in Settings
